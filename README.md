# READ ME - Tutorial
A simple arcade game built in Python using PyGame for AS92004 (Year 11 Digital Technologies).

The player moves left and right to dodge falling rocks. The game gets harder as your score increases.

---

## How to Install and Run

### Step 1 — Make sure Python is installed
Open your terminal and type:
```
python --version
```
You should see something like `Python 3.x.x`. If you get an error, download Python from https://www.python.org/downloads/

---

### Step 2 — Install PyGame
In your terminal type:
```
pip install pygame
```
Wait for it to finish. You should see `Successfully installed pygame`.

---

### Step 3 — Download the files
Click the green **Code** button on this page, then click **Download ZIP**. Extract the folder somewhere on your computer.

---

### Step 4 — Open in VS Code
1. Open VS Code
2. Click **File → Open Folder**
3. Select the DodgeTheRocks folder

---

### Step 5 — Run the game
Click on the file you want to run in the left panel, then press **F5** or click the **Run** button (▶) in the top right.


---

## How to Play

- Press **LEFT arrow** to move left
- Press **RIGHT arrow** to move right
- Dodge the falling rocks to score points
- You lose a life each time a rock hits you
- Game over when all lives are gone

### Excellence version only
- When the game starts, press **1** for Easy, **2** for Normal, **3** for Hard
- After game over, press **R** to replay or **Q** to quit


---

## Requirements

- Python 3.x
- PyGame (`pip install pygame`)
